What is Ploinky?

Ploinky is an open-source platform that revolutionizes how developers work with containerized environments and AI agents. It serves as an intelligent wrapper around Linux containers, providing a unified interface for managing both local CLI tools and cloud-deployed functions.

Quick Start

Installation

# Clone the repository
git clone https://github.com/PlonkyRepos/ploinky.git
cd ploinky

# Add to PATH
export PATH="$PATH:$(pwd)/bin"

# Or create system-wide links (optional)
sudo ln -s $(pwd)/bin/ploinky /usr/local/bin/ploinky
sudo ln -s $(pwd)/bin/p-cli /usr/local/bin/p-cli

Prerequisites

  • Node.js 18 or higher
  • Docker or Podman
  • Git

First Commands

# Start Ploinky interactive mode
ploinky

# List available agents
ploinky> list agents

# Run a bash agent
ploinky> run agent bash

# See available repositories
ploinky> list repos

# Add more agents
ploinky> add repo cloud

Shared Architecture

Both Ploinky CLI and Cloud share a common foundation that ensures consistency and portability:

üìù Manifest-Based Configuration

Each agent is defined by a simple JSON manifest that specifies its container, commands, and metadata.

üîê Container Isolation

All agents run in isolated containers, ensuring security and reproducibility across environments.

üåç GitHub Distribution

Agents are distributed through GitHub repositories, making sharing and version control seamless.

üîì Open Source

Fully open-source with MIT license, ensuring no vendor lock-in and complete transparency.
